### 三、功能描述
系统分为（身份）：普通用户(患者)、医生、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 出诊医生
5. 公告信息
6. 留言板
7. 个人中心
8. 我的收藏




**医生功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 出诊医生管理
5. 预约挂号管理
6. 预约取消管理
7. 病例信息管理
8. 药品信息管理
9. 处方开具管理



**管理员功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 患者管理
5. 医生管理
6. 科室管理
7. 出诊医生管理
8. 预约挂号管理
9. 预约取消管理
10. 病历信息管理
11. 药品信息管理
12. 处方开具管理
13. 留言板管理
14. 系统管理（轮播图管理、公告信息）
